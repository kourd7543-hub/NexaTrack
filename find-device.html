How Device Recovery Works

1

Before Losing Device

Go to home page → click "Track My Location" → click "Share Location". This generates a unique device code.

2

Save Your Device Code

Note the TS-XXXXX code or save the share link in a safe place like your email.

3

If Device is Lost

Come to this page from any device. Enter your device code or phone number to find it.

4

Via Phone Number

Enter the lost device's phone number. A WhatsApp/SMS message with tracking link is sent. When opened, location is shared.
