 Auto-Track — Setup Before You Lose Your Phone

Install Tasker on your Android phone and add the config below. If your phone ever gets stolen or lost, it will silently send its GPS location to NexaTrack every few minutes — even if the SIM is changed.

Install Tasker from Play Store
New Task → HTTP POST action
Paste URL + Body below
Profile → Time → Every 15 min
https://nexa-track.vercel.app/api/location
{
 "lat": "%LOC".split(",")[0],
 "lon": "%LOC".split(",")[1],
 "accuracy": "%LOCACC",
 "time": "%DATE %TIME",
 "device": "%DEVID",
 "link": "https://maps.google.com/?q=%LOC"
}
Content-Type header set karo: application/json  |  Also add Get Location action before HTTP Request, Source: Any, Timeout: 30s  |  Battery optimization: Tasker ke liye Unrestricted set karo
