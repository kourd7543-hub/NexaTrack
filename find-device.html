Lost Device Finder

Choose a recovery method based on your situation

Enter Your Device IMEI

Dial *#06# on your phone to find IMEI. Save it somewhere safe before losing device.

IMEI blocking permanently disables the device on all Indian networks. Only use if device is truly stolen/lost.

Block via CEIR Portal

Government of India portal to block stolen phones across all networks.

  • 1 Visit ceir.gov.in
  • 2 Click "Block Stolen/Lost Mobile"
  • 3 Enter IMEI + mobile number
  • 4 Upload FIR copy
  • 5 Submit — device blocked in 24hrs
Open CEIR Portal

File Police Complaint

File FIR online or at nearest police station. Required for CEIR blocking.

  • 1 Go to your state police portal
  • 2 File online FIR / complaint
  • 3 Mention IMEI number in complaint
  • 4 Get FIR copy (needed for CEIR)
  • 5 Keep complaint number safe
Cybercrime Portal

Contact Your Carrier

Block SIM to prevent misuse of your number.

  • 1 Call carrier helpline immediately
  • 2 Request SIM block
  • 3 Get duplicate SIM same number
  • 4 Airtel: 121 | Jio: 198 | Vi: 199
  • 5 BSNL: 1800-180-1503

Secure Your Accounts

Immediately secure all accounts accessible from lost device.

  • 1 Change Google account password
  • 2 Sign out all devices from Google
  • 3 Change banking app passwords
  • 4 Enable 2FA everywhere
  • 5 Revoke WhatsApp linked devices

Auto-Track Setup

Setup Tasker on your phone before losing it. If lost, it will automatically send location to NexaTrack every few minutes — even if someone changes SIM.

01

Install Tasker

  • Download Tasker from Play Store (paid app ~₹350)
  • Grant all permissions: Location, Background, Battery
  • Disable battery optimization for Tasker
  • Settings → Apps → Tasker → Battery → Unrestricted
02

Create New Task

  • Open Tasker → Tasks tab → "+" button
  • Name it: SendLocation
  • Add Action → Net → HTTP Request
  • Method: POST
  • URL: (copy from box below)
03

Set Location Variables

  • Before HTTP Request, add: Get Location action
  • Source: Any
  • Timeout: 30 seconds
  • Body (JSON): (copy from box below)
  • Content-Type header: application/json
04

Create Profile (Trigger)

  • Profiles tab → "+" → Time
  • Every: 15 minutes (repeat)
  • Link to Task: SendLocation
  • OR add trigger: "Power disconnected" for theft alert
  • Enable profile — green tick shows it's active

Your Tasker Config Values

HTTP Request URL

https://nexa-track.vercel.app/api/location

Request Body (JSON)

{
  "lat": "%LOC".split(",")[0],
  "lon": "%LOC".split(",")[1],
  "accuracy": "%LOCACC",
  "time": "%DATE %TIME",
  "device": "%DEVID",
  "link": "https://maps.google.com/?q=%LOC"
}

Setup this before losing your phone. Once lost, come back here and use "Quick Find" tab → Watch Live Location to see the map update automatically.
